COC 1 - Course Overview

This section focuses on the basics of computer system assembly and installation.

Lesson 1: Assembly and Disassembly of Computer Unit

In this lesson, we will go over the basic steps to disassemble and reassemble a computer unit.

Step-by-Step Procedure

  1. Prepare the necessary tools (screwdrivers, anti-static wrist strap, etc.).
  2. Disassemble the computer by opening the case and removing internal components (RAM, HDD, etc.).
  3. Assemble the computer by placing the components back in their respective slots.
  4. Test the computer and ensure everything is working, including BIOS settings.

Lesson 2: Prepare Bootable USB

This lesson teaches how to create a bootable USB drive using Rufus for OS installation.

Step-by-Step Procedure

  1. Download Rufus and install it.
  2. Prepare a USB drive (at least 8GB) and choose it in the Rufus interface.
  3. Select the Windows ISO file and the appropriate partition scheme (GPT for UEFI).
  4. Click Start to create the bootable USB.
