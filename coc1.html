COC 1 - Course Overview

Lesson 1: Assembly and Disassembly of Computer Unit

Step-by-Step Procedure

  • Prepare the necessary tools (screwdrivers, anti-static wrist strap, etc.).
  • Disassemble the computer by opening the case and removing internal components (RAM, HDD, etc.).
  • Assemble the computer by placing the components back in their respective slots.
  • Test the computer and ensure everything is working, including BIOS settings.

Lesson 2: Prepare Bootable USB

Step-by-Step Procedure

  • Download Rufus software and install it.
  • Plug in a USB drive and open Rufus.
  • Select the bootable OS ISO file and configure settings.
  • Click Start to create the bootable USB.
